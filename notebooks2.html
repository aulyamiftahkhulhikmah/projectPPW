Preprocessing PTA dan LDA Modelling#

Preprocessing#

Preprocessing data teks adalah tahapan penting dalam analisis teks dan pemrosesan bahasa alami (Natural Language Processing/NLP).Preprocessing adalah proses pra-pemrosesan data yang melibatkan serangkaian langkah untuk membersihkan, mengubah, atau menyusun data agar lebih sesuai atau dapat diolah dengan lebih baik oleh model atau algoritma tertentu. Tujuan dari preprocessing adalah untuk meningkatkan kualitas data, mengurangi noise, dan mempersiapkannya agar cocok untuk analisis atau pemodelan lebih lanjut. Langkah-langkah dalam preprocessing dapat mencakup pembersihan data, penghapusan atau pengelolaan nilai yang hilang, normalisasi, transformasi fitur, dan langkah-langkah lainnya sesuai kebutuhan spesifik tugas atau analisis data.

Beberapa langkah preprocessing umum dalam NLP melibatkan:

  • Pembersihan Teks (Text Cleaning): Menghilangkan karakter khusus, tanda baca, dan elemen lain yang tidak diinginkan yang tidak memberikan informasi berarti untuk analisis.

  • Case folding: membersihkan teks dalam kolom ‘Abstrak’. Ini menghapus karakter-karakter yang tidak relevan seperti tanda baca, mengubah teks menjadi daftar kata-kata dan menghilangkan karakter khusus seperti tanda baca, simbol, angka atau karakter yang tidak relevan.

  • Tokenisasi: Memisahkan teks menjadi unit-unit yang lebih kecil, yang biasanya disebut sebagai “token.” Token bisa berupa kata, frasa, atau karakter, tergantung pada tingkat granularitas yang diinginkan.

  • Stopword Removal: Menghapus kata-kata umum (stopwords) yang tidak memberikan kontribusi signifikan pada analisis karena keberadaannya yang umum dalam bahasa.

  • Stemming atau Lemmatisasi: Mereduksi kata-kata ke bentuk dasar mereka (lemmatisasi) atau memotong akhiran kata (stemming) untuk mengurangi variasi kata yang memiliki akar kata yang sama.

Case Folding#

Case Folding digunakan untuk membersihkan teks dalam kolom ‘Abstrak’. Ini menghapus karakter-karakter yang tidak relevan seperti tanda baca, mengubah teks menjadi daftar kata-kata dan menghilangkan karakter khusus seperti tanda baca, simbol, angka atau karakter yang tidak relevan.

import re, string

# Text Cleaning
def cleaning(text):
    # Menghapus tag HTML
    text = re.compile('<.*?>|&([a-z0-9]+|#[0-9]{1,6}|#x[0-9a-f]{1,6});').sub('', str(text))

    # Mengubah seluruh teks menjadi huruf kecil
    text = text.lower()

    # Menghapus spasi pada teks
    text = text.strip()

    # Menghapus Tanda Baca, karakter spesial, and spasi ganda
    text = re.compile('<.*?>').sub('', text)
    text = re.compile('[%s]' % re.escape(string.punctuation)).sub(' ', text)
    text = re.sub('\s+', ' ', text)
    text = re.sub(r'\n', ' ', text)
    text = re.sub("â", "", text)

    # Menghapus Nomor
    text = re.sub(r'\[[0-9]*\]', ' ', text)
    text = re.sub(r'[^\w\s]', '', str(text).lower().strip())
    text = re.sub(r'\d', ' ', text)
    text = re.sub(r'\s+', ' ', text)

    # Mengubah text yang berisi 'nan' dengan whitespace agar nantinya dapat dihapus
    text = re.sub('nan', '', text)

    return text

results = []

    # Membagi data menjadi train dan test set
    X_train, X_test, y_train, y_test = train_test_split(X_lda, y, test_size=0.2, random_state=42)
